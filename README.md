# 🎬 Scene*Writer

シーン構成型シナリオエディター - ボイスドラマや音声作品のシナリオ執筆に特化したWebアプリケーション

## 📦 含まれるツール

### 💻 Scene*Writer PC版
デスクトップ・ノートパソコン向けのフル機能版。大画面で快適に執筆作業ができます。

### 📱 Scene*Writer スマホ版  
スマートフォン・タブレット向けに最適化。外出先でもアイデアをすぐに形にできます。

### 🔄 データ変換ツール
Voice×Writerとの相互変換や、旧バージョンのデータ移行をサポートします。

## 🚀 使い方

### オンラインで使用する場合
以下のURLにアクセスしてください：
```
https://[あなたのユーザー名].github.io/scene-writer/
```

### ローカルで使用する場合
1. このリポジトリをダウンロード
2. 各フォルダ内の`index.html`をブラウザで開く

## 📁 ファイル構成

```
scene-writer/
├── index.html           # メインページ（ツール選択画面）
├── README.md           # このファイル
├── pc/                 # PC版
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── assets/
├── mobile/             # スマホ版
│   ├── index.html
│   ├── style.css
│   └── script.js
└── converter/          # データ変換ツール
    ├── index.html
    ├── style.css
    └── converter.js
```

## ✨ 主な機能

- **シーン管理**: 最大300シーンまでの管理が可能
- **キャラクター管理**: 登場人物の登録とワンクリック挿入
- **自動文字数カウント**: リアルタイムで文字数と推定時間を計算
- **効果音・BGM**: 音声演出の指定が簡単に
- **データ互換性**: PC版とスマホ版でデータ共有可能
- **オフライン動作**: 一度読み込めばインターネット接続不要

## 🖥️ 動作環境

- **対応ブラウザ**: Chrome, Firefox, Edge, Safari（最新版推奨）
- **必要な機能**: JavaScript, localStorage
- **推奨画面サイズ**: 
  - PC版: 1280×720以上
  - スマホ版: 5インチ以上

## 📝 データ形式

- **プロジェクトファイル**: .vwp形式（JSON）
- **エクスポート**: .txt形式（プレーンテキスト）
- **Voice×Writer互換**: 変換ツール使用で相互変換可能

## 🔧 トラブルシューティング

### プロジェクトが開けない
- ファイル形式が.vwpまたは.jsonか確認してください
- ブラウザのJavaScriptが有効になっているか確認してください

### 文字数が更新されない
- ブラウザをリロード（F5キー）してみてください

### データが保存されない
- ブラウザのlocalStorageが有効になっているか確認してください
- プライベートブラウジングモードでは保存されません
